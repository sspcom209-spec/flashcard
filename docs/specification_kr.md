# 스페인어 플래시카드 웹 앱 — 소프트웨어 명세서

## 개요

스페인어 어휘 학습을 돕는 프런트엔드 전용 플래시카드 웹 애플리케이션입니다. **Vite**, **React**, **TypeScript**를 사용하여 개발되었습니다. 이 앱은 **정적 데이터**를 사용하여 카테고리별로 그룹화된 플래시카드를 표시합니다. 학습, 퀴즈, 복습 모드를 지원하며, 브라우저 내 상태를 통해 성능을 추적합니다.

---

## 기술 스택

- **프런트엔드 프레임워크:** React (Vite + TypeScript 사용)
- **데이터 저장:** 정적 JSON/TypeScript 파일 (백엔드 없음)
- **상태 관리:** React의 `useState` / `useReducer`, 선택적으로 `useContext` 또는 Zustand 사용
- **스타일링:** CSS Modules, TailwindCSS 또는 원하는 CSS 전략
- **데이터 영구 저장:** 선택적으로 localStorage 사용 (오답 또는 통계 저장용)

## 기능

### 1. 플래시카드 학습 모드

- 플래시카드는 기본적으로 스페인어 단어를 보여줍니다.

- 카드를 클릭하면 카드가 뒤집혀 영어 번역이 나타납니다.

- 카드를 뒤집으면 두 개의 버튼이 나타납니다.

- ✅ "맞았습니다"

- ❌ "틀렸습니다"
- 앱은 틀린 카드를 기억합니다.

### 2. 다시 풀기 모드

- 이전 라운드에서 **"틀림"**으로 표시된 카드만 학습할 수 있는 옵션입니다.

- 학습 모드와 동일하게 작동하지만, 틀린 카드만 포함됩니다.

- 사용자는 언제든지 이 목록을 초기화할 수 있습니다.

### 3. 카테고리

- 카드는 다음과 같은 카테고리로 분류됩니다.

- **동물**

- **음식**

- **동사**
- 사용자는 학습 또는 퀴즈를 풀 카테고리를 선택할 수 있습니다.

### 4. 퀴즈 모드

- 두 가지 유형의 퀴즈:

- **객관식**

- 스페인어 단어 1개, 영어 보기 4개 (정답 1개 + 오답 3개)

- 오답 보기는 문제별로 고정되어 있습니다.

- **빈칸 채우기**

- 스페인어 단어가 제시되면 사용자가 영어 단어를 입력합니다.

- 대소문자를 구분하지 않고 정확히 일치해야 합니다.

### 5. 통계 페이지

- 다음 통계를 추적합니다(브라우저 메모리 또는 localStorage 사용):

- 카테고리별 학습 카드 수

- 정답 수 vs. 오답 수

- 정확도(%)

## UI 페이지/구성 요소

### 1. 홈 페이지

- 환영 메시지
- 다음 페이지로 이동하는 버튼/링크:

- 학습 모드

- 퀴즈 모드

- 통계 페이지

### 2. 카테고리 선택 페이지
- 사용 가능한 카테고리 표시: 동물, 음식, 동사
- 모드 선택 (학습/퀴즈)
- 플래시카드로 이동 또는 퀴즈

### 3. 플래시카드 학습 구성 요소
- 한 번에 한 장씩 카드 표시
- 카드 넘기기 애니메이션 (선택 사항)
- "정답" 및 "오답" 버튼

### 4. 다시 실행 구성 요소
- 플래시카드 학습과 동일한 UI를 사용하지만, 틀린 카드만 표시

### 5. 퀴즈 구성 요소
- 객관식 또는 주관식
- 각 문제 후 피드백 제공 (정답/오답)

### 6. 통계 페이지
- 모든 세션 요약
- 데이터 영구 저장을 위한 LocalStorage 지원 (선택 사항)
